[ 애디스콥 구조 ]
애디스콥은 메인 ( 인터페이스 + 코어 )과
네트워크사 마다 하나의 추가적인 팟( = 프로젝트, 어댑터 )로 존재함.
A,D,E 네트워크사의 광고를 넣을 경우, 해당 3개를 추가적으로 팟 인스톨하는 개념

[ 애디스콥 광고 ]
애디스콥 메인에는 오퍼월 광고만 있음 ( 설치하고 리워드 받는 무료충전소 )
다른 어댑터들로 Rewarded Vidoe와 Interspitial 광고가 있음
리워디드는 영상보고 템 받는거고,
인터스피셜은 화면 전반에 뿅 띄우고 닫히는거임.
