# OKR 
Objective & Key Results

[ 추상적인 설명 ]
OKR은 구글이 처음 사용하고 성공한 이유라고 여기저기 공표하여,
큰 회사들이 따라서 도입한, 
현재 유행하는 조직 업무 관리 도구이다.

[ 작성법 정의 ]
O (Objective) 는 목표에 대한 추상적인 설명이며,
KR (Key Results) 는 이것을 달성했는지 알려주는 구체적인 설명, 수치이다.

예시 )
Objective : 이전에 없었던 새로운 시작은 개척한다.
Key Result 1 : 20대 신규 고객 2만명
Key Result 2 : 신규 서비스의 매출 20억
Key Result 3 : 신규 서비스의 고객 만족도 4.8

[ 작성법 추가 설명 ]
Objective는 회사의 미션을 달성하는 것과 구성원들이 진심으로 원하는게 무엇인지를 조화하여 작성해야한다.
"이번 분기 매출 20억" 보다는, "모든 애견인이 좋아하는 브랜드로 알려진다" 라는 식으로.

Key Results는 목표를 달성했다는 구체적인 수치, 이들이 다 달성되었을 때 Objective가 정말 달성되는지, 현재 상태에서 열심히 했을 때 70% 정도  달성 가능한 정도로 작성하는 것이 좋다.
